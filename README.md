# PHP版本的数据库访问框架 #

## 项目依赖 ##
无。

## 使用说明 ##
 * 可以直接使用数据库访问的实现类，也可以使用IOC框架注入数据层对象。
 * sqlite数据库的访问，可以配置SQLITEDATABASE常量，定义数据库的路径。
